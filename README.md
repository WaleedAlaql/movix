# Movix 🎬

A modern, high-performance Flutter movie application that showcases trending and top-rated movies using the TMDB API. Built with a clean architecture and robust state management.

---

## Features

* **Trending Movies:** Real-time update of daily trending cinema directly from TMDB.
* **Top Rated Section:** Seamlessly fetch and view the highest-rated movies of all time.
* **Detailed View:** Cinematic movie details page featuring descriptions, global ratings, dynamic release dates, and languages.
* **Clean UI/UX:** Dark-themed, immersive cinema experience built with fluid scrolling and reusable custom widgets.

## 🛠️ Architecture & Tech Stack

This project follows **Clean Software Engineering principles**, separating the data layer, business logic, and UI components to ensure scalability and ease of testing.

* **Framework:** Flutter (Dart)
* **State Management:** [Riverpod](https://riverpod.dev/) (with modern Code Generation syntax `@riverpod`)
* **Networking:** HTTP client for robust API communications.
* **Data Parsing:** Type-safe JSON serialization using automated Dart data models (`factory constructors`).<img width="385"
